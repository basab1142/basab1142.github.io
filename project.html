Image Captioning

In this project I have used VGG-16 to extract feature from a image and fed it into a Sequential layer of Embedding and LSTM

This is a kaggle challenge (flickr8k) containing around 8k set of data. Each corresponding one or more caption. Model

Facial Emotion recognition

Here I have used 3 convolution layer followed by a maxpooling layer to extract data and then 2 dense layer having dropout layer in between

As the classes are ['anger', 'surprise', 'disgust', 'fear', 'neutral', 'happiness', 'sadness', 'contempt'] an output layer with 8 neuron having softmax function
